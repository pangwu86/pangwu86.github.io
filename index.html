自媒体新手期

发表于 2017-12-28 | | 阅读次数

头条号

  • 新手期每天一篇文章，视频无限量。
  • 转正需要发布10篇标记已推荐的文章（视频），并且头条指数达到650。
  • 头条指数每周发布一次
  • 可以定时发布，也可以先存草稿

网易号

  • 自12月4日新功能上线后，新入驻平台的账号将经历新手期阶段。平台将会根据账号新手期内创作情况不定期审核通过账号新手期。在新手期结束后，同样根据账号新手期内的创作表现予以综合评定，赋予账号相应积分。
  • 每天可发6篇文章，视频

吐槽：视频不支持FLV格式, 而且转码特别慢
吐槽：不能存草稿
吐槽：不能定时发

百家号

  • 新手期每天一篇文章，视频
  • 可以同步头条号
  • 可以存草稿，不能定时发

大鱼号

  • 每天可发3篇文章，视频
  • 百分制，达到75分开始进入转正流程
  • 大鱼是UC，优酷，土豆都发

腾讯号

  • 每天可发6篇文章，视频

关于文章

  • 标题要明确含义
  • 可以故意写错，然后引发用户吐槽，增加评论数

iOS踩坑记录001-误删Assets.xcassets后如何恢复

发表于 2017-09-01 | | 阅读次数

删除后，Xcode没有undo操作，就从回收站里手动copy出来放回原位，但是编译后发现图标的设置不起作用了，但也没有相关错误提示。后来经过测试发现需要2个步骤

  1. 删除Xcode中看到那个文件，选择“Remove Refrence”，也就是删除引用
  2. 在项目文件夹上右键，选择“Add Files to Project”，选择刚才的文件夹

这样再编译就正常了。

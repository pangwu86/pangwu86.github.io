iOS学习笔记001-批量生成图标

发表于 2017-09-01 | | 阅读次数

由于要适配多种设备多种分辨率，iOS会要求你生成一堆大小不一的图标，人工做这么一批肯定是要命了，必须找到合适的工具才能事半功倍。

这里推荐下图标工程http://icon.wuruihong.com/, 除了iOS其他平台需要的图标也可以自动生成，实在是居家旅行必备工具呀。

直接进入Xcode，删除原项目Assets.xcassets中的AppIcon.appiconset，然后把生成的拖进去就好了

软件(包)管理工具

发表于 2017-12-22 | | 阅读次数

nvm

官网：https://github.com/creationix/nvm

Node版本管理工具，可以方便的在多个node版本间切换

查看本地安装的node

nvm ls

查看远程可安装的node

nvm ls-remote

安装大版本最新node

nvm install 9

安装大版本最新node并重新安装全局package

nvm install 9 –reinstall-packages-from=8

设置某版本为默认版本

nvm alias default v9.3.0

homebrew

官网：https://brew.sh/

MasOS下的包管理工具，可以安装各种常见软件比如Nginx，mysql等

更新homebrew

brew update

更新所有安装包

brew upgrade

列出所有安装包

brew list

安装某个包

brew install nginx

更新指定安装包

brew upgrade nginx

查看可清理的过期包

brew cleanup -n

清理过期包

brew cleanup

我爱烹饪(1)

发表于 2017-03-05 | | 阅读次数

之前一直做饭但是总是做一些凑合吃的家常菜，过年期间迸发出一个想法，那就是好好练手艺然后过年给家人做一桌年夜饭。

然后就是各种尝试新菜，发现自己天赋值不错，学起来非常容易，所以就有了下面这些作品。

这是一月底到三月初的作品，希望以后可以持续更新下去，不断的丰富自己的“菜谱”。

口水鸡

做了两次味道很ok，就是麻芝酱放到的有点多那个汤汁太稠了

咕咾肉

买了菠萝就尝试做了，加了辣椒后的改版

酸汤酥肉

味道棒棒的

鸡汤面

用煮口水鸡的汤汁下的面条，那叫一个鲜！

情人节特制蛋包饭

这个蛋皮做起来最麻烦，还是在孙姐姐的帮助下搞定的

还有寄宿一个月的小球球出镜，真单身狗！

吉野家牛肉饭

模仿吉野家的牛肉饭，简单好做又好吃的典范。

酸辣笋尖粉

济南就没吃到一家好吃的桂林米粉，最终大淘宝拯救了me，以后不求人了

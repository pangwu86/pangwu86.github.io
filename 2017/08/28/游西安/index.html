游西安

发表于 2017-08-28 | | 阅读次数

时隔8年，再次来到了十三朝古都西安。这次是全家三口一起出动，蘑菇同学也是第一次坐飞机，一路上虽然捣蛋无数，但终归玩的不错没出啥乱子。

由于带娃出动，行程不可避免的被缩短，基本只有半天时间出门，每天也就能逛一个地点。好在住的酒店就在市中心，周边不远的回民街(北广济街，北院门，西羊市，化觉巷，洒金桥)逛了个遍，泡馍，biangbiang面，贾三包子，肉夹馍，酸汤水饺等著名美食也尝了个遍，基本上在吃的方面可以说不虚此行了。

至于景点么，也就逛了个古城墙和大雁塔，蘑菇同学还在城墙上哭了个溜够，应该说一个可以玩泡泡，玩玩具的地方或许更适合她现在的年龄。

时间虽短，但是出来逛逛吃吃喝喝放松以下总是好的，带娃出门的经验这次可以说是levelup了，毕竟一个人带娃坐飞机这种高难度动作不是谁都搞的定的^_^

还有那个西安买的宝宝尿盆，本来都扔了，被打扫房间的服务员专程给me们拿了下来，最后飞机带回家了，只能说命中注定无法抛弃你😆

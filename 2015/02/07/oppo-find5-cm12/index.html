Find5刷CM12

发表于 2015-02-07 | | 阅读次数

这几天学习android开发(第二次了, 上一次的忘光光了), 突然想尝试下5.0, 就查了一下CM的支持列表, 惊喜的发现竟然还有Find5, 这下老机又可以焕发青春了

oppo的find5是2年前发布的”老机器”了, 不过1080p屏幕等硬件指标放到今天还能算过的去, 说明作为那时候的旗舰机还是用料挺足的.

find5算是国产机最开放的了, 可以各种刷rom. 自家的ColorOS, CM, 老罗的SmartisanOS等等. 都试过一圈后(那时候还是CM11), 发现还是自家的孩纸自己最清楚, 也最用心, 折腾n久后还是回到了ColorOS的怀抱, 因为用起来最舒服, 省心.

这次刷机到CM12(android 5.0.2), 可以说还是挺眼前一亮的. 当然这更多的是归功于android的大版本带来的新特性, ART, Material Design等等, 上手2天来明显觉得用起来比以往的都爽快. 当然CM还是处于nightly, 本地化都没做完(CM11就全汉化好了), 各种闪屏, 小bug频现, 但瑕不掩瑜, 手动加上Gapps后真是用起来各种舒坦, 再也不想换回国产那些rom了, 特别是一想到哪些各家应用商店就恶心的受不鸟.

说一下刷机步骤, 其实是从oppo的论坛里找来的, 自己动手整合了下

  1. 刷recovery
  2. 对data分区进行扩容, 5.0.x只有ART模式了, 安装文件后占的磁盘容量比之前大了很多, 2GB的data分区远远不够了
  3. 进recovery -> 双清 -> 刷最新的cm包, 记得进入提前把各种包放在/sdcard中
  4. 遇到”基带未知”, 刷底包fix_radio.zip
  5. 遇到”未root”, 刷SuperSU_Pro_v2.40-Final_update_iKiraCN.zip
  6. 遇到”想要google的那些软件”, 刷gapps, 这里是适合5.0.x的
  7. 遇到“在wifi或网络上的感叹号”, 直接执行
    adb shell su -c “settings put global captive_portal_server g.cn”

再放个上面的刷机资源合集, me都下好了

晒几张图吧

  1. 各种扁平化, 各种鲜艳的色彩使用

  2. 图标排的好满

  3. 好像就锤子快捷按钮有VPN, 这可是国人必备呀, 不然要点5-6下才能打开vpn. 不过那个ADB的网络开关真是为了开发而设计的呀

  4. 常用的搜狗会crash, 看来5.0的还是有些兼容性问题, 最后选了G家的拼音输入法, 配色布局真是好搭

  5. 中英混杂, 各种地方都有

  6. 能感受上最新的android, 要感谢CM和oppo

  7. 还一份净土, 远离国产应用商店, 没一个好鸟

最后再说一句, find5用了2年了, 质量真心不错, 国货良心产品.

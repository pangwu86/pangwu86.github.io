如何将多看里的笔记转为Markdown格式

发表于 2015-03-11 | | 阅读次数

用多看看书有个好处就是可以直接将笔记导出到Evernote(印象笔记)里

但是导出的格式却不方便引用到自己的文章中, 所以需要做一点小小的转换, 方法很简单就是一行用于替换的正则表达式


    // 查找
    [\d]{4}-[\d]{2}-[\d]{2} [\d]{2}:[\d]{2}:[\d]{2}\n

    // 替换
    \n<br/>\n>


接下来看看疗效如何^_^

  1. 从Evernote里选中要转换的笔记

  2. 找个文本编辑器贴一下刚才选中的内容

  3. 使用给出的正则查找替换下

  4. 转换后的为markdown格式

  5. 最后在页面中的效果

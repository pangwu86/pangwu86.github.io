用多看看书有个好处就是可以直接将笔记导出到Evernote(印象笔记)里

但是导出的格式却不方便引用到自己的文章中, 所以需要做一点小小的转换, 方法很简单就是一行用于替换的正则表达式


    // 查找
    [\d]{4}-[\d]{2}-[\d]{2} [\d]{2}:[\d]{2}:[\d]{2}\n

    // 替换
    \n<br/>\n>

Find5刷CM12

发表于 2015-02-07 | | 阅读次数

这几天学习android开发(第二次了, 上一次的忘光光了), 突然想尝试下5.0, 就查了一下CM的支持列表, 惊喜的发现竟然还有Find5, 这下老机又可以焕发青春了

oppo的find5是2年前发布的”老机器”了, 不过1080p屏幕等硬件指标放到今天还能算过的去, 说明作为那时候的旗舰机还是用料挺足的.

find5算是国产机最开放的了, 可以各种刷rom. 自家的ColorOS, CM, 老罗的SmartisanOS等等. 都试过一圈后(那时候还是CM11), 发现还是自家的孩纸自己最清楚, 也最用心, 折腾n久后还是回到了ColorOS的怀抱, 因为用起来最舒服, 省心.

这次刷机到CM12(android 5.0.2), 可以说还是挺眼前一亮的. 当然这更多的是归功于android的大版本带来的新特性, ART, Material Design等等, 上手2天来明显觉得用起来比以往的都爽快. 当然CM还是处于nightly, 本地化都没做完(CM11就全汉化好了), 各种闪屏, 小bug频现, 但瑕不掩瑜, 手动加上Gapps后真是用起来各种舒坦, 再也不想换回国产那些rom了, 特别是一想到哪些各家应用商店就恶心的受不鸟.

说一下刷机步骤, 其实是从oppo的论坛里找来的, 自己动手整合了下
